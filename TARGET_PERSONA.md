# 目標店型設定（內部用）

## Type A：生活雜貨・選品店
**類似店型**：台隆手創館、日本 LOFT、Hands Beams
**店面性格**：商品種類極雜，從文具到美妝到廚房用品，陳列道具需要高度通用性＋快速換檔彈性
**他們頭痛的事**：
- 商品尺寸差異太大，標準層板/掛勾永遠不對
- 換檔期等於換一批陳列，耗時又耗成本
- 百貨櫃位限制多，道具不能釘牆、不能超寬
**採購決策者**：商品部的人（女性居多），他們在乎的不是價格，是「這個道具能不能讓我下週換檔少加班兩天」
**碧豐能做的事**：可拆式層板掛勾系統，木質＋金屬混搭，讓雜貨店也能有精品感

## Type B：運動用品・戶外裝備
**類似店型**：日本 ALPINE、Sports Depot、Mont-bell
**店面性格**：商品又大又重（滑雪板、登山鞋、帳篷），陳列道具的強度比外觀更重要
**他們頭痛的事**：
- 掛勾撐不住重物，層板變形
- 顧客每天摸來摸去，鍍層半年就掉
- 季節性商品換裝（滑雪季→露營季）需要整店重擺
**採購決策者**：商品本部的人（男性居多，本身也是運動愛好者），他們在乎的是「這個掛勾能不能撐住我店裡最重的商品」
**碧豐能做的事**：金屬沖壓＋焊接，高荷重掛勾，18K 金等級的精密加工用在這裡叫「耐久性」

## Type C：眼鏡鏡框店
**類似店型**：JINS、OWNDAYS、眼鏡市場、ALOOK、Zoff
**店面性格**：商品量不多但單價高，陳列是銷售的核心——顧客不是來買眼鏡，是來「試戴體驗」
**他們頭痛的事**：
- 眼鏡展示掛勾的通用性（不同鏡框寬度/重量/材質）
- 防盜需求（高單價商品，安全掛勾必備）
- 鏡框反覆拿取，掛勾磨損快
- 顧客視角：眼鏡要好看→展示架本身不能搶戲
**採購決策者**：VM（視覺陳列）負責人＋商品部，他們在乎的是「這個展示架能不能讓顧客不自覺地拿起眼鏡試戴」
**碧豐能做的事**：眼鏡專用防盜掛勾（已有日本眼鏡品牌實績）、細緻金屬表面處理、展示架＋燈光整合

## Type D：藥妝店（松本清型 / @cosme 型 / 藥局型）
**類似店型**：松本清、@cosme store、OS Drug、SUNDRUG、ツルハドラッグ、Tomod's、日藥本舖

### 走進店裡你會看到什麼
一進門，右手邊是**開架美妝試用區**——一整排帶燈的化妝檯，鏡子周圍有一圈燈泡，檯面上擺滿了試用品，每個試用品旁邊站著一個小小的壓克力牌。檯面下方是庫存抽屜，拉開來整整齊齊放著同一款商品的不同色號。

往前走是**壁面陳列架**——從地板到天花板，密密麻麻的陳列架，層板是透明的壓克力板或者細鐵網，每層高度剛好可以放兩排商品。層板邊緣有一條細細的價格軌道（price rail），上面插著一張張的價格卡。

**走道中間**有獨立的中島櫃（gondola），兩面都可拿商品，端頭（end cap）永遠擺著當期促銷品——可能是一整箱的面膜堆疊，或者一個壓克力陳列盒裝著旅行組。

**結帳櫃檯前**有一排小小的衝動購買區——口香糖、小條護唇膏、OK 繃，放在一個斜面的小展示架上，讓排隊的人順手拿。

### 這些店用什麼樣的陳列道具
| 道具類型 | 長什麼樣 | 用在哪裡 |
|---------|---------|---------|
| 壁面層板 | 透明壓克力板或細鐵網，寬 90cm，可調高度 | 主力陳列區，放保養品/藥品 |
| 價格軌道（Price Rail） | 鋁合金或塑膠條，卡在層板前緣 | 每層層板前面，插價格卡 |
| 試用化妝檯 | 帶燈鏡子 + 檯面 + 下方抽屜，尺寸約 120x60cm | 美妝開架區 |
| 中島櫃（Gondola） | 雙面可拿，鐵製或木製，底部有腳墊 | 走道中間 |
| End Cap 展示架 | 壓克力盒或鐵架，擺在走道端頭 | 促銷區 |
| 衝動購買架 | 小型斜面架，約 30x30cm | 結帳櫃檯前 |
| 試用座（Tester Stand） | 小壓克力座，放試用品＋標籤 | 美妝區檯面上 |
| 面膜陳列盒 | 壓克力或紙盒，一格一格的，像抽屜分隔 | 壁面或中島 |
| 掛勾（Hooks） | 金屬掛勾，勾在洞洞板或槽板上 | 壁面，掛面膜/小物 |
| 分類籃（Bins） | 塑膠或鐵線籃，可堆疊或掛在架上 | 促銷品/特價區 |

### 藥妝店的三種子類型

#### 1. 大型連鎖藥妝（松本清型）
- 200-500 坪，都在車站前或商店街
- 商品從美妝到藥品到零食都有
- 陳列需求：**大量標準化層板＋中島櫃**，換檔頻率高
- 採購者：本社 MD（50 歲上下），重視 cost performance
- 他們頭痛：每次展店要訂 50 組以上的層板＋中島櫃，但每家店尺寸微調很麻煩

#### 2. 美妝選品店（@cosme store 型）
- 50-150 坪，都在百貨公司
- 只賣美妝保養品，而且是「精選過的」
- 陳列需求：**質感比數量重要**，燈光整合、試用區是靈魂
- 採購者：VM 負責人（30-40 歲女性），重視視覺一致性
- 他們頭痛：日本總部要求海外店跟日本店長一模一樣，但台灣的工班做不出來

#### 3. 社區藥局（藥局型）
- 20-50 坪，都在住宅區
- 藥品為主，美妝為輔，還有一些保健食品
- 陳列需求：**功能性強**，藥品分類清楚，處方櫃檯動線順
- 採購者：藥局店主或店長，自己決定
- 他們頭痛：空間小，要怎麼在有限的坪數內放最多的商品

### 共通痛點（所有藥妝店）
1. **商品替換頻率高**— 美妝品每季出新色，陳列道具要跟得上
2. **試用品管理**— 試用座被摸髒、被偷、乾掉，陳列設計要讓試用區好維護
3. **價格標籤更換**— 紙本價格卡換到手軟，而且容易掉
4. **衛生法規**— 藥品區不能跟食品混放，需要明確的分區標示
5. **防盜**— 高單價美妝/藥品，需要防盜陳列

### 採購決策者畫像
- **日本總部 MD**（50-60 歲男性）：決定年度展店計劃，在意的是「供應商能不能穩定供貨」
- **海外事業部開發**（40-50 歲）：執行展店，在意的是「當地的工班能不能做出我要的品質」
- **VM 負責人**（30-40 歲女性）：決定陳列細節，在意的是「這個道具能不能讓商品看起來更好賣」
- **店長**（40-50 歲）：實際使用的人，在意的是「好不好清潔、好不好換價格卡」

### 碧豐的切入點
- 藥妝店最需要的就是**標準化＋少量彈性調整**的系統——碧豐的一站式整合正好符合
- 試用化妝檯的帶燈鏡子、壓克力陳列盒、金屬掛勾——碧豐的異材質整合能力全包
- 日系品管標準，日本藥妝品牌可以直接信任
- 已經有日本客戶實績（眼鏡品牌），藥妝品牌的採購邏輯類似

## Type E：其他潛在跨國連鎖
**類似店型**：
- 台灣餐飲海外展店（鼎泰豐、春水堂）→ 設備統一規格，全球出貨
- 亞洲選品店（Pinkoi 實體、誠品）→ 小量但高品質的 Bespoke 需求
- 歐美精品飾品連鎖 → 精緻展示、防盜、品牌一致性

**共通特性**：少量多樣、跨國展店、需要一個能統包且品質穩定的供應商
**碧豐的位置**：這些品牌不想同時管理十家工廠，他們想找到一個「可以溝通、品質穩定、願意接小單」的夥伴
